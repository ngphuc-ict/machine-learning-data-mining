# Machine Learning and Data Mining Course Project

This repository contains assignments, experiments, and project implementations for the **Machine Learning and Data Mining** course.

## Project Objectives

The project focuses on building, training, and evaluating various machine learning and deep learning models for data analysis and prediction tasks.

Main topics covered include:

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Baseline MLP Model
* Hyperparameter Tuning
* Convolutional Neural Networks (CNN)
* Transfer Learning
* Recurrent Neural Networks (RNN/LSTM)
* Transformer Models
* Failure Analysis and Model Evaluation
